## Getting Started

### Prerequisites

- Node v18.18.0 or higher

### Installation

#### Clone the repo

```shell
git clone https://github.com/hardworking-toptal-dev/dust.git
```
#### Install Dependencies

```shell
npm install
```

#### Start the web app

```
npm run dev
```
